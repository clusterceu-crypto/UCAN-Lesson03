# UCAN Lesson 03 — Case Source Handoff 1.3.3

## Використання

Цей файл фіксує learner-facing кейси сторінки 7 та межі допустимих тверджень для HTML Builder, QA і наступних патчів.

## CASE-L03-UA-01 — Львів

- **Місце:** Львівська міська територіальна громада, Україна.
- **Тема:** відкриті дані громадського транспорту.
- **Підтверджена дія:** оприлюднення GTFS Realtime vehicle positions і GTFS Static.
- **Підтверджені дані:** позиції транспорту, маршрути, зупинки та довідкова інформація; ресурс позицій оновлюється приблизно кожні 11 секунд.
- **Офіційне джерело:** https://data.gov.ua/dataset/lviv-public-transport-gtfs-real-time
- **Допустимий learner-facing висновок:** стандартизовані дані створюють основу для інформування, аналізу маршрутів та контролю послуги.
- **Не стверджувати без додаткового джерела:** конкретне скорочення затримок, витрат, викидів або скарг.

## CASE-L03-INT-01 — Роттердам

- **Місце:** Роттердам, Нідерланди.
- **Тема:** 3D-модель міста і просторова база даних.
- **Підтверджена дія:** муніципалітет відкрив 3D-модель міста для повторного використання.
- **Підтверджені дані:** базові реєстри будівель, адрес і громадського простору; LiDAR; стереофотограмметрія; тематичні 3D-сервіси.
- **Офіційне джерело:** https://www.rotterdam.nl/rotterdam-in-3d
- **Допустимий learner-facing висновок:** спільна просторова модель може бути базою для аналізу й узгодження проєктів.
- **Не стверджувати без додаткового джерела:** виміряне скорочення витрат, термінів проєктування або викидів.

## CASE-L03-INT-02 — Амстердам

- **Місце:** Амстердам, Нідерланди.
- **Тема:** цифрові карти кліматичної адаптації.
- **Підтверджена дія:** створено інтерактивні карти спеки, посухи, водонавантаження та ризику затоплень на основі кліматичних стрес-тестів.
- **Підтверджене використання:** результати стрес-тестів використовуються під час формування стратегії кліматичної адаптації.
- **Офіційне джерело:** https://maps.amsterdam.nl/klimaatadaptatie/
- **Допустимий learner-facing висновок:** просторові дані допомагають бачити нерівномірність ризиків і готувати пріоритетні рішення.
- **Не стверджувати без додаткового джерела:** конкретне зменшення температури, збитків або кількості підтоплень.

## Presentation rule

Кейс подається у згорнутій картці з полями:

1. проблема;
2. що зроблено;

4. управлінська користь;
5. що може використати українська громада;
6. офіційне джерело.
