# UCAN Lesson 03 — Patch 1.3.1

## Призначення

Контрольований патч додає пілотний компонент **Learning Transition Layer** між завершенням теорії та Interactive Self Check.

## Замінити файли

- `index.html`
- `css/style.css`
- `js/script.js`

Файл `js/lesson03-ai-prompts.js` включено без зміни learner-facing AI prompts.

## Архітектура після патчу

1. Вступ
2–6. Теорія
7. Learning Transition Layer
8. Самоперевірка
9. Практика
10. Портфель мера
11. Тест
12. Завершення

## Додано

- короткі приклади застосування даних у громадах;
- learner-facing міжнародні орієнтири;
- чотири офіційні ресурси з Patch 1.3;
- два офіційні відео з Patch 1.3;
- необов’язкову нотатку перед практикою з локальним збереженням;
- передачу цієї нотатки до AI Assistant як додаткового контексту;
- семантичне визначення сторінки тесту замість жорстко заданого номера.

## Не змінено

- теоретичні тексти;
- практичну форму та Portfolio;
- AI prompts;
- тест і правильні відповіді;
- Test Gate;
- затверджені Visual Assets;
- наявні localStorage keys.

## QA

Виконано статичну перевірку HTML, JavaScript syntax check, перевірку 12 сторінок, унікальності HTML ID, порядку сторінок, семантичного Test Gate, зовнішніх посилань, accessibility-атрибутів і перенесення всіх ресурсів та відео.

Статус: **Ready for Functional QA**.
