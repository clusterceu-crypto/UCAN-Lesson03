# UCAN Lesson 03 — Patch 1.3.2

## Призначення

Контрольований патч коригує пілотний компонент **Learning Transition Layer** відповідно до evidence boundary курсу.

Непідтверджені узагальнені «приклади громад» вилучено. Конкретне місце, дія та результат можуть з’являтися у learner-facing матеріалі лише разом із перевіреним офіційним джерелом.

## Замінити файли

- `index.html`
- `css/style.css`
- `js/script.js`

Файл `js/lesson03-ai-prompts.js` включено без змін для повного комплекту патчу.

## Що змінено

- вилучено три узагальнені картки «Як це працює в українських громадах»;
- вилучено дві узагальнені картки «Корисний міжнародний досвід»;
- сторінка одразу переходить до розділу «Де взяти перевірені поради»;
- ресурси та відео подано у компактних розгортаних блоках `details/summary`;
- карта заняття оновлена відповідно до нового змісту сторінки;
- стартовий прогрес виправлено з 9% на 8% для 12-сторінкової архітектури;
- заголовок першої сторінки отримав програмний фокус;
- повідомлення про локальне збереження нотатки тепер враховує помилку `localStorage`;
- уточнено текст підтвердження скидання прогресу.

## Не змінено

- теоретичні тексти сторінок 1–6;
- самоперевірку, практичну форму та Portfolio;
- learner-facing AI prompts;
- тест, правильні відповіді та Test Gate;
- затверджені Visual Assets;
- наявні localStorage keys;
- чотири офіційні ресурси та два відео Patch 1.3.
