# UCAN Lesson 03 — Patch 1.3.3

## Призначення

Контрольований патч перебудовує сторінку **«Перед тим як перейти до практики»** за сильною моделлю сторінки 5 Заняття 02:

1. конкретні міста й підтверджені дії;
2. окремі орієнтири для поглиблення;
3. окремий блок відеолекцій курсу «Циркулярна економіка»;
4. компактне розкриття інформації через `details/summary`.

## Замінити файли

- `index.html`
- `css/style.css`

Файли `js/script.js` і `js/lesson03-ai-prompts.js` включено для повного контрольованого комплекту; learner-facing AI prompts не змінено.

## Конкретні міські приклади

### Львів, Україна

Офіційний набір відкритих даних громадського транспорту у форматах GTFS Realtime та GTFS Static. Картка показує, як стандартизовані й регулярно оновлювані дані створюють основу для інформування, аналізу маршрутів та перевірки транспортної послуги.

### Роттердам, Нідерланди

Офіційна відкрита 3D-модель міста на основі базових реєстрів, LiDAR і стереофотограмметрії. Картка показує цінність спільної просторової основи до запуску складної цифрової платформи.

### Амстердам, Нідерланди

Офіційна інтерактивна кліматична карта з шарами спеки, посухи, водонавантаження та ризику затоплень. Картка показує зв’язок між стрес-тестами, просторовими даними та стратегією адаптації.

## Окремі блоки сторінки

- **Як це працює у конкретних містах** — три згорнуті картки.
- **Орієнтири для поглиблення** — чотири офіційні інструменти C40.
- **Відео курсу «Циркулярна економіка»** — дві відеолекції, визначені CE Video Integration Map для UCAN-L03.
- **Маленьке завдання** — локальна нотатка, яка передається AI Assistant як додатковий контекст.

## Відео L03

- Лекція 3-1 «Загальна характеристика відкритих даних, принципи та міжнародні стандарти».
- Лекція 1-5 «Розвиток інфраструктури для циркулярної економіки».

## Не змінено

- теоретичні тексти сторінок 1–6;
- 12-сторінкову архітектуру;
- самоперевірку, практичну форму та Portfolio;
- learner-facing AI prompts;
- тест, правильні відповіді та Test Gate;
- localStorage keys;
- затверджені Visual Assets;
- чотири офіційні ресурси для поглиблення.

## Evidence boundary

Картки не приписують містам непідтверджені кількісні результати. Вони фіксують лише конкретне місце, офіційно описану дію, тип даних, управлінську можливість і переносний урок для української громади.
